Descripción del Proyecto
Sistema inteligente de rutas para Zipaquirá basado en aprendizaje supervisado. El proyecto utiliza datos de trayectos entre diferentes estaciones para entrenar un modelo capaz de clasificar y recomendar las rutas más eficientes según criterios de tiempo, fluidez y comodidad.
